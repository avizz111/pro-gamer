computer's parts


Monitor (In computing, a monitor, also called a screen, computer monitor, and computer monitor, is the main output device, displaying data or information to all users. It can also be considered an input/output peripheral if the monitor contains a touch or multi-touch screen.)

Keyboard (In computing, a keyboard is an input device, partly inspired by the typewriter keyboard, which uses a system of stitches or margins to act as mechanical levers or electronic switches that send all information to the computer or to the mobile phone.)

cpu (The CPU (Central Processing Unit) is the main component of the computer for processing instructions. A CPU receives input from the user, the operating system, applications, and software programs.)

mouse (The mouse is part of the peripherals of your equipment and is the part of the computer that allows you to interact with the objects that appear on the screen, through a cursor or pointer that you will see on the monitor.)

speakers (They are the devices that give audio output to the computer, thanks to them you can listen to the sound of the music or video that you are playing. Depending on the model, you can connect them to the USB or audio port. Some computers have built-in speakers inside the monitor.)

printer (Machine that is connected to an electronic computer and that serves to print the selected information contained in it.)

camera (A webcam or network camera is a small digital camera connected to a computer which can capture images and transmit them over the Internet, either to a web page or other computers privately. A webcam needs a computer to transmit the images.)

Operating sistem (An operating system is the set of programs of a computer system that manages hardware resources and provides services to software application programs. These programs are executed in privileged mode with respect to the rest.)
